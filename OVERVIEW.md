## 🚀 QUICK START GUIDE

### 1️⃣ Install (5 minutes)
```bash
npm install
cd server && npm install && cd ..
```

### 2️⃣ Configure (5 minutes)
```bash
# Edit .env.local
REACT_APP_GITHUB_USERNAME=your_username

# Edit server/.env
SMTP_USER=your_email@gmail.com
SMTP_PASS=your_app_password
```

### 3️⃣ Run (2 minutes)
```bash
npm run dev
```
